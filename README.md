NFL QB Contract Value Analysis
-------------------
Project Overview
-------------------
This project analyzes quarterback contracts in the National Football League by comparing player salary with performance metrics to evaluate whether top-paid quarterbacks deliver value relative to their peers.

The analysis focuses on financial efficiency by measuring performance output relative to contract cost across quarterbacks in similar salary tiers.


Background
--------------------------
The project was initially created to evaluate whether Jalen Hurts provides strong contract value relative to other quarterbacks receiving top-tier salaries.

The analysis tests this hypothesis by comparing quarterback performance metrics against contract value across players in comparable salary brackets.

The data used was is from the season of 2024-2025 (The sseason that the project was made after)

Objective
-------------------
Determine whether highly paid quarterbacks deliver performance proportional to their salary relative to their peers.

Data & Metrics
-------------------------
The analysis compares several key financial and performance indicators:

Quarterback salary

Total yards (passing + rushing)

Touchdowns

Efficiency ratings

Cost-per-performance metrics

Tools Used
--------------------
Software: Microsoft Excel

Excel Tools

Data Validation – Created dropdown menus to dynamically select quarterbacks.

Charts & Graphs – Built bar charts to compare QB financial efficiency vs league averages.

Conditional Formatting – Highlighted player performance relative to league averages.

Interactive Dashboard Design – Built a visual dashboard displaying player stats, financial metrics, and comparisons.

Pivot Tables – Used to summarize and aggregate player statistics.

Excel Functions / Formulas
--------------------------
XLOOKUP – Retrieved player statistics, team data, and financial metrics dynamically.

INDEX / MATCH – Performed advanced lookups across multiple conditions.

IF Statements – Created logic for performance comparisons and efficiency calculations.

AVERAGE – Calculated league averages for benchmarking.

SUM / SUMIF – Aggregated player statistics and totals.

Custom Financial Metrics
----------------------------
Several financial efficiency metrics were created to evaluate contract value:

Cost per Win

Cost per Yard

Cost per Touchdown

Adjusted Cost per Win (penalizing losses and availability)

Win Efficiency

Availability Efficiency

Data Analysis Techniques

The project applies several analytical techniques including:

Performance benchmarking vs league averages

Salary-to-performance financial modeling

Sports analytics performance evaluation

Data visualization and dashboard reporting

Conclusion
------------

Based on the metrics analyzed, Jalen Hurts demonstrates performance output consistent with or exceeding peers in the same compensation range.
